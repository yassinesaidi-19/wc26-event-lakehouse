# Databricks Skeleton

This directory is a migration-ready scaffold for moving the local tournament lakehouse into Databricks.

Suggested execution order:

1. `01_ingestion.py`
2. `02_event_log.py`
3. `03_canonical_model.py`
4. `04_state_engine.py`
5. `05_marts_quality.py`

The files are placeholders for notebook/task decomposition, not a finished Databricks deployment.
